Laravel AI SDK 引入 sub-agent 支持
x.com · Taylor Otwell (X)
Laravel AI SDK 刚刚支持了 sub-agent,意味着一个父级 agent 可以在自己的 tools() 里返回另一个 agent,让它去处理专门子任务。这个设计在架构上很优雅:每个 sub-agent 有自己的指令、工具、provider 配置和独立的 context window,父 agent 只负责调度不负责执行。对于 Laravel 生态的 AI 应用来说,这意味着可以构建真正分层的 AI 工作流,而不是把所有逻辑塞进一个 prompt 里。这是 PHP 生态少有的、值得其他框架参考的 AI 架构设计。
